Zig University

Learn the Zig programming language through practical examples

About Zig University

Zig University is a comprehensive collection of examples and tutorials designed to help you learn the Zig programming language. Our goal is to provide clear, practical examples that demonstrate Zig's powerful features and capabilities.

Getting Started

To get started with these examples, you'll need:

  • Zig 0.14.0 or compatible version
  • Basic understanding of programming concepts
  • A text editor or IDE

Clone this repository to run the examples locally:

git clone https://github.com/codenlighten/zig-university.git
cd zig-university

Featured Examples

Memory Allocators

Explore Zig's various memory allocation strategies, including the General Purpose Allocator, Arena Allocator, and more.

zig run 01_memory_allocators.zig

Compile-Time Features

Discover Zig's powerful compile-time metaprogramming capabilities, including type reflection and compile-time function execution.

zig run 10_comptime_features.zig

Foreign Function Interface (FFI)

Learn how to seamlessly interface with C code from Zig, demonstrating Zig's excellent C ABI compatibility.

cd 13_ffi && zig build run

Learning Path

For those new to Zig, we recommend following this learning path:

  1. Start with the basic examples (Memory Allocators, Data Structures)
  2. Move on to error handling and testing concepts
  3. Explore file I/O and string manipulation
  4. Dive into threading and concurrency
  5. Study the more advanced topics like compile-time features and FFI
